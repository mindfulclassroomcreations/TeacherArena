🚀 NEXT STEPS - GET STARTED IN 15 MINUTES!

IMPORTANT: You need to install Node.js first!

1. Install Node.js
   → Using Homebrew: brew install node
   → Or download from: https://nodejs.org/

2. Install dependencies
   → Run: npm install

3. Set up Supabase (database)
   → Create account at https://supabase.com
   → Run SQL from: supabase-schema.sql
   → Copy your credentials

4. Get OpenAI API key
   → Sign up at https://platform.openai.com
   → Create API key
   → Add credits ($5 minimum)

5. Configure environment
   → Copy: cp .env.example .env.local
   → Add your API keys to .env.local

6. Start the app
   → Run: npm run dev
   → Open: http://localhost:3000

🆘 NEED HELP?

Troubleshooting:
  1. Check INSTALLATION-GUIDE.md troubleshooting section
  2. Review CHECKLIST.md to ensure all steps completed
  3. Check browser console for errors
  4. Review Supabase/OpenAI dashboards for issues

Common Issues:
  • "npm: command not found" → Install Node.js
  • "Failed to fetch ideas" → Check Supabase credentials
  • "Failed to generate ideas" → Check OpenAI API key
  • "Port 3000 in use" → Kill process: lsof -ti:3000 | xargs kill -9
